Best Minimal restaurant is one of the easiest to use restaurant themes. It is a simple and unique restaurant theme.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Go to Appearance -> Install plugins, and install/activate the recommended plugins.
5. Now you are ready to customize your theme style and content.

== Frequently Asked Questions == 

How do I publish started content?

1. To publish starter content, you will need to install this theme on a fresh WordPress installation, then go to theme page "Appearance >> Customize", and then click on Publish button at the top of the page.
2. This will upload the images, create all the pages, add navigation menus, and set site options.
3. The next step is to go through each page, change the starter content to your own content and then Save each page.
